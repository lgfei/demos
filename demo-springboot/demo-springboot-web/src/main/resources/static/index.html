学习笔记

Spring Boot 项目目录结构(Maven) src/main/java：java源代码文件
src/main/resources：配置文件
src/main/resources/templates：模板页面文件
src/main/resources/static：是静态资源文件目录，在这个目录中的所有文件将可以被直接访问，如果没有这个文件夹可自行创建
src/main/resources/public：作用和src/main/resources/static目录一样。
src/test/java：目录下放置测试代码
src/test/resources：测试代码相关配置
什么是YAML文件? YAML是JSON的一个超集，也是一种方便的定义层次配置数据的格式
Springboot默认支持两种配置文件类型：.properties和.yml
它们作用是一样的，只是写法不一样而已
可以同时存在，如果存在重复配置，优先取.properties的配置
@Controller和@RestController的区别? @RestController注解相当于@ResponseBody ＋ @Controller合在一起的作用。
意思就是用@RestController标识后，controller里面的方法都以json格式输出 ，return的结果直接写入HTTP response body中，不会被解析为跳转路径。
